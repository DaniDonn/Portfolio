OBJECTIVE:

  • Automate a system for operators to request supervisor assistance efficiently.
  • Provide both a visual alert (flashing light) and an automated email notification.
  • Build the system using a Raspberry Pi with a pushbutton, relay, and light, all coded in Python.

PROCESS:

  • Button & Light Test: Verified hardware functionality by activating a relay that turns on the light when the button is pressed.
  • Supervisor Information Setup: Created a text file storing:
    • Supervisor contact info
    • Their shift times
    • Sender email credentials
  • Data Parsing:
    • Wrote a parser to read the text file and store each lineâ€™s info in a tuple (to handle mixed data types).
    • Determined the correct supervisor based on current time vs. shift times.
  • Email Automation:
    • Used Selenium WebDriver to access Gmail.
    • Composed the email using MIMEMultipart and MIMEText.
    • Sent the email automatically via the smtplib module.

RESULT:

  • Successfully built a low-cost alert system that sends real-time emails and activates a flashing light for location visibility.
  • Improved plant communication and reduced supervisor response time.
  • Strengthened Python skills and introduced automation using a Raspberry Pi.
  • Learned to integrate hardware interaction with email and time-based logic.
