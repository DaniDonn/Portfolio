FANUC Welding Simulation

OBJECTIVE:

  • Understand how to work with RoboGuide and program a FANUC robot using TCP and work objects.
  • Run the tip of a welding tool in a circular path on the faces of two cylinders.
  • Create three concentric paths with 50mm separation.
  • On a flat plate, create a square pattern of lines: 10 lines in X and 10 lines in Y, each 500mm long, spaced 50mm apart.
  • Pick the weld tool tip off of the plate by 50mm between paths.
  • The final result is demonstrated in the video below.

PROCESS:

  • Created 7 positions: start, above first cylinder, center of first cylinder, above second cylinder, center of second cylinder, flat plate, and end position.
  • Used point registers to store data and manipulate robot movements.
  • Set up three user frames: one for each cylinder and one for the flat plate, simplifying programming.
  • Wrote 83 lines of code to execute the task.
  • Lines 1-5: Set up user frame, tool, and moved to start position.
  • Lines 6-9: Created four point registers to facilitate circle creation.
  • Lines 13-19: Calculated and executed circular paths using loops and register values.
  • Repeated circle creation process on the second cylinder after switching user frames.
  • Lines 44-62: Used loops and registers to create 10 vertical weld lines on the flat plate.
  • Lines 63-81: Used a similar approach to create 10 horizontal weld lines.
  • Final step: Robot moved to position 7, marking the end of the task.

RESULT:

  • Successfully programmed the FANUC robot to follow precise welding patterns.
  • Utilized user frames and point registers to simplify complex movements.
  • Executed smooth transitions between different sections of the task.
  • Final welding pattern accurately met project specifications.
