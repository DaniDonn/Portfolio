OBJECTIVE:

  • Design and simulate a robot work cell for palletizing using ABB RobotStudio.
  • Develop a program to efficiently stack cylindrical cans into a specified box layout.
  • Utilize proximity sensors and limit switches for precise operation.

PROCESS:

  • Cell Layout and Hardware:
    • Placed the ABB robot near the in-feed conveyor belt and box to minimize travel distance.
    • Used an E3ZM-T83 proximity sensor to detect incoming cans, pausing the conveyor until each can is picked.
    • Incorporated a limit switch to detect when the box is full, triggering a green signal light.
    • Used an MG10 electromagnetic gripper as the end effector for reliable can handling.
  • Program Logic:
    • Created nested FOR loops to stack 24 cans in a 2-layer pattern.
    • Utilized the MoveJ and MoveL commands for efficient point-to-point and linear movements.
    • Implemented the pickup() and dropoff() functions to control the magnet, including wait timers for precise grip control.
    • Integrated position tracking using the OverExactPointOnBox and CanDropoff coordinate variables to handle row and layer transitions.
  • Error Handling and Efficiency:
    • Included checks for X and Y direction limits to properly transition between rows and layers.
    • Used structured loops and spacing logic to ensure optimal placement without collisions.
