Tab Detection

OBJECTIVE:

  • Develop a computer vision system to detect unbuffed tabs on machined parts to improve quality control and reduce defects.
  • Automate defect detection to reduce manual inspection time and improve overall efficiency in production lines.

PROCESS:

  • Data Collection & Model Training:
    • Captured over 100 labeled images of machined parts, including various lighting conditions and orientations.
    • Created training and testing datasets, using approximately 80% of images for training and 20% for testing.
    • Trained the model using Darknet, an open-source neural network framework that implements the YOLO (You Only Look Once) model for real-time object detection.
  • Detection Optimization:
    • Identified that small tabs were difficult to detect, leading to the development of a Python script to divide the image into four quadrants for closer analysis.
    • Implemented quadrant-based zoom to enhance the model's ability to detect small tabs, significantly improving detection accuracy.
  • Planned Conveyor Integration:
    • Designed a plan for integrating the camera system with a Raspberry Pi-controlled conveyor belt for automated inspection.
    • Planned for error signaling, including flashing lights to alert operators to defects.

RESULT:

  • Successfully trained a convolutional neural network to identify tabs on machined parts with high accuracy.
  • Improved defect detection capabilities, reducing the risk of defective parts reaching downstream processes.
  • Gained hands-on experience in machine learning, computer vision, and real-world defect detection.
