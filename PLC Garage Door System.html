OBJECTIVE:

  • Design a system that can safely and automatically open or close a garage door using multiple input options.
  • Ensure the system accounts for safety, emergency scenarios, and operational limits.

PROCESS:

  • Users can control the garage door with two keypads or a remote FOB using red and green square buttons:
    • Green button: opens the garage door
    • Red button: closes the garage door
  • All input devices also include a circular red button to immediately stop the garage door in motion.
  • Each open/close action counts as one cycle. After 8 cycles, the system is disabled and a red light appears, signaling required maintenance.

RESULT:

  • Emergency stop (E-stop) buttons to immediately halt motor operation
  • Seal-in contacts in ladder logic to prevent motor restart after power outage
  • Emergency override switch to allow manual opening even during car charging
  • Photoelectric sensor to detect obstacles under the garage door and trigger a reverse/open command with a flashing yellow light
  • System successfully simulates real-world garage door behavior with cycle limits, safety stops, and fault-handling logic.
  • Project documentation available for more technical details: Automatic Garage Door System Final Project
