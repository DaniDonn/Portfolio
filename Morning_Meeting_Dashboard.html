OBJECTIVE:

  • Develop an interactive SCADA dashboard using Ignition Perspective, Python scripting, and CSS.
  • Create a user-friendly interface accessible to all employees, with limited edit access for select users.
  • Replace the traditional whiteboard approach to streamline daily manufacturing meetings and improve data tracking.

PROCESS:

  • User Access Control:
    • Set up user authentication to limit edit access while keeping the dashboard visible to all employees.
  • Table Setup and Data Entry:
    • Created multiple tag-linked tables for real-time data entry, including machine status, engine production, shipping, and cost.
    • Designed a note pad section for unstructured entries that didnâ€™t fit into predefined tables.
  • Calendar Design for Safety Tracking:
    • Built a dynamic, button-based calendar with 31 buttons to track daily safety incidents.
    • Implemented color-coded updates (green for no accidents, red for accidents) for quick visual reference.
  • Interface Optimization:
    • Used CSS for professional styling and responsiveness, ensuring compatibility across various screen sizes.
    • Incorporated dropdown menus to streamline data input and reduce screen clutter.
    • Organized content with embedded views to create a clean and efficient layout.

RESULT:

  • Developed a comprehensive dashboard that replaced the need for physical whiteboards in morning meetings.
  • Enhanced real-time visibility into production metrics, reducing the need for manual data handling.
  • Provided a scalable and professional tool for tracking manufacturing floor performance.
