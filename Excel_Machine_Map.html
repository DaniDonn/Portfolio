OBJECTIVE:

  • Develop a user-friendly Excel tool to help employees locate machines within the plant.
  • Reduce confusion and time spent searching by visually mapping machine locations.
  • Provide functionality similar to a mall directory map using Excel and Visual Basic.

PROCESS:

  • User Interface:
    • User types a machine name into a search bar and clicks the search button.
    • Results display the machine name and corresponding plant cell.
    • A highlighted rounded rectangle appears on the plant map showing the machine location.
  • Search by Cell:
    • Typing a cell name displays all machines in that cell with their locations highlighted on the map.
  • “Go To Location” Button:
    • Zooms in on the highlighted rectangles for easier visibility.
  • Backend Data:
    • A dedicated data sheet contains:
      • Cell names
      • Machine numbers
      • X and Y coordinates on the map
      • Date last checked
      • A completion status symbol
  • Coordinate Mapping:
    • Created a Visual Basic tool to drag the highlight shape and click a button to display its exact coordinates.
    • Used these coordinates to accurately map machines on the layout.
  • Visual Basic Programming:
    • Coded all functionality using Visual Basic for Applications (VBA), including:
      • Search and match logic
      • Shape highlighting and movement
      • “Clear All” and “Go To Location” buttons

RESULT:

  • Fully functional plant map search tool built entirely in Excel with VBA.
  • Successfully helped employees find machine locations quickly and efficiently.
  • Learned and self-taught Visual Basic for this project, building all logic independently.
  • Enhanced understanding of coordinate-based UI and dynamic shape manipulation in Excel.
