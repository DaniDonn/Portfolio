  • Develop a semi-automated system using a Universal Robot and Cognex 7802 vision camera to collaborate with operators in assembling encapsulated fuel cells.
  • Ensure accurate rivet placement and heat staking using camera-based detection and robotic coordination.
  • Design and fabricate custom hardware components to support the automation process.
  • Camera Calibration & Communication:
    • Calibrated a Cognex 7802 camera using a dot grid to accurately identify the "best placed" rivet in the tray.
    • Aligned the camera and robot coordinate systems by creating a new reference plane on the tray using the robot arm.
  • Coordinate Conversion & Pick and Place:
    • Used the pose_trans function to convert coordinates from the rivet tray plane into the robotâ€™s base coordinate system.
    • Programmed the robot to:
      • Move to the rivet location
      • Slowly approach and activate a vacuum to pick it up
      • Move and place the rivet on a cathode plate fixture
      • Repeat until all four rivets were placed using a loop
  • Heat Staking & Control Logic:
    • Programmed a loop where the robot heat staked each rivet using an "until force detected" condition for precise control.
    • Integrated a pressure sensor to detect if a rivet was still in the pickhead after placement, confirming whether it was successfully released.
    • Built logic based on a switch that allowed operators to select either a left or right plate, which determined placement patterns.
    • Added prompts at the end of each cycle to ask the operator whether to repeat the process or re-home the robot if switching plates.
  • Mechanical Design:
    • Modeled the rivet tray, the dual pick and place head, and the heat staker adapter in SolidWorks using proper GD&T.
    • Rapid prototyped the components to verify fit and function.
    • Final components were machined after confirming prototypes were too flimsy for long-term use.
  • Delivered a fully functional semi-automated system that reliably collaborated with operators to assemble fuel cells.
  • Resolved coordinate transformation issues between the camera and robot for accurate motion control.
  • Improved system reliability and safety with pressure sensor feedback and operator-driven program branching.
  • Successfully designed and implemented custom hardware that supported consistent and repeatable performance.
